# LoggerJS

[![NPM Version][npm-image]][npm-url]


A logger wrapper around winston.

<br>

---

## Usage


Basic usage out of the box:
```javascript
import Logger from '@potentii/logger-js';

Logger.info('PROCESS_FAILED', 'Processing has failed', { userId: '...', processType: '...' });

```
